Atributos

A seguir tem uma lista de atributos e uma Explicação para o que cada um serve:

  • Class="..." Este atributo da uma classe a uma tag e serve praticamente classificar tags e a mesma classe pode ser dada a mais de uma tag.
  • id="..." faz a mesma coisa que o atributo class só que ele da um id a uma tag e esse id é único, ou seja não tem como ter duas tags com o mesmo id.
  • Style="..." Com este atributo podemos incluir elementos CSS na tag(Não é recomendado usar o método inline).
  • long="..." Define o idioma do elemento.
  • title="..." Define o titulo do elemento.
  • alt="..." Define um texto alternativo (Normalmente é utilizado na tag img).
  • hidden="..." Oculta o elemento
  • align="..." Você define o padrão de alinhamento.(right,center,left,justify)
  • width="..." Define a largura
  • height="..." Define a altura
